# Tasky

A simple task management application

# Technologies

- TanStack
- Axios
- Ant.Design
- Firebase

# To Run Locally

1. Clone the repo

```
git clone https://github.com/434huzaifa/Tasky
```

2. Go to the cloned directory

```
cd Tasky
```

3. Install dependencies

```
npm i
```

4. Run the project

```
npm run dev
```


Enjoy
